#+TITLE: Cuckoo Hashing
#+AUTHOR: Subramanya J
#+OPTIONS: toc:nil

Uses two hash tables with two hashing functions to ensure O(1) worst case lookup time.

* Building
Requires gcc.
Use the Makefile with the make command to build.
